Stack: NodeJS, ReactJS, JavaScript, PostgresQL, CSS
Description: This application is for those who like to play pickup games but have trouble finding a group to do so.

You can post, join, edit, and cancel sporting events. When you join the event it will add it to your calendar on your account. We have a chat feature for people to decide roles for the event as well as a map to show where the event is relative to you. This project really helped me learn to work in a group.
